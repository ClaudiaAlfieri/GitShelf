# 📚 GitShelf

A GitHub repository explorer built with **React** as part of a React learning journey. Search, save and explore your favourite GitHub repositories, with issues filtering and pagination, all in one place 🔍.

## 🚀 Features

- 🔎 Search and save GitHub repositories
- 💾 Persistent storage with localStorage
- 📋 View open issues for each repository
- 🏷️ Filter issues by state (All, Open, Closed)
- 📄 Paginated issue listing
- 🗑️ Remove saved repositories
- ⚡ Live data from the GitHub API

## 🛠️ Tech Stack

- ⚛️ React
- 🧭 React Router DOM
- 🎨 Styled Components
- 🐙 GitHub REST API
- 🎯 React Icons
- 📦 Axios

## 📦 Installation

```bash
git clone https://github.com/ClaudiaAlfieri/GitShelf.git
cd GitShelf
npm install
npm start
```

The app will open at `http://localhost:3000` 🚀

## 🎯 How to Use

### Saving Repositories:
- 🔎 Type a repository in the format **owner/repo** (e.g. `facebook/react`)
- ➕ Click the **+** button to add it to your shelf
- 🗑️ Click the **trash icon** to remove a repository

### Exploring Issues:
- 📋 Click the **menu icon** on any saved repository to view its issues
- 🏷️ Use the filter buttons to switch between **All**, **Open** and **Closed** issues
- ⬅️ ➡️ Navigate between pages using **Voltar** and **Próxima**
